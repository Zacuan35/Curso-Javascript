Sintaxis de JavaScript

La sintaxis de JavaScript es el conjunto de reglas, cómo se construyen los programas de JavaScript:

var x, y, z; // Declare Variables
x = 5; y = 6; // Asignar valores
z = x + y; // Calcular valores



Valores

La sintaxis de JavaScript define dos tipos de valores:

  • Valores fijos: se denominan Literales
  • Valores variables: se denominan Variables

Literales de JavaScript

Existen dos tipos de literales, los números y las cadenas.

  • Números
  • Cadenas

Y las dos reglas de sintáxis más importantes para estos valores fijos son:

Números Cadenas
1. Los números se pueden escribir con o sin decimales:

10.50
1001

Ejemplo

document.getElementById("demo").innerHTML = 10.50;

2. Las cadenas son texto, escrito entre comillas simples o dobles:

"Mario"
'Mario'

Ejemplo

document.getElementById("demo-2").innerHTML = 'Mario';

Variables de JavaScript

En un lenguaje de programación, las variables se utilizan para almacenar valores de datos.
JavaScript usa la palabra clave var para declarar variables.
Se utiliza un signo igual para asignar valores a las variables.

En este ejemplo, x se define como una variable. Entonces, a x se le asigna el valor de 6:

Ejemplo

var x;
x = 6;
document.getElementById("demo-3").innerHTML = x;

Operadores

Operadores aritméticos Operadores de asignación

JavaScript usa operadores aritméticos ( + - * /) para calcular valores:

Ejemplo

document.getElementById("demo-4").innerHTML = (5 + 6) * 10;

JavaScript usa un operador de asignación (=) para asignar valores a las variables:

Ejemplo

var x, y;
x = 5;
y = 6;
document.getElementById("demo-5").innerHTML = x + y;

Expresiones

Una expresión es una combinación de valores, variables y operadores, que se calcula en un valor.
El cálculo se llama evaluación.
Por ejemplo, 5 * 10 se evalúa como 50

Las expresiones pueden contener valores literales y variables.

Valores Literales Valores Variables

Valor con números

document.getElementById("demo-6").innerHTML = 5 * 10;

Valor con variables

var x;
x = 5;
document.getElementById("demo-7").innerHTML = x * 10;

Por ejemplo, "Pepe" + "" + "Wuicho", se evalúa como "Pepe Wuicho":

"Pepe" + " " + "Wuicho"

Valor con cadenas

document.getElementById("demo-8").innerHTML = "Pepe" + " " + "Wuicho";

Palabras clave

Las palabras clave de JavaScript se utilizan para identificar acciones.
La palabra clave var le dice al navegador que cree variables:

Ejemplo

var x, y;
x = 5 + 6;
y = x * 10;
document.getElementById("demo-9").innerHTML = y;

Comentarios

No todas las declaraciones de JavaScript se "ejecutan".
El código después de barras dobles // o entre /* y */ se trata como un comentario.
Los comentarios se ignoran y no se ejecutarán:

Ejemplo

var x;
x = 5; // Seré ejecutado
// x = 6; No seré ejecutado
document.getElementById("demo-10").innerHTML = x;


Aprenderá más sobre los comentarios en un capítulo posterior.

Identificadores

Los identificadores son nombres.
En JavaScript, los identificadores se utilizan para nombrar variables (y palabras clave, funciones y etiquetas).
Las reglas para los nombres legales son muy parecidas en la mayoría de los lenguajes de programación.
En JavaScript, el primer carácter debe ser una letra, un guión bajo (_) o un signo de dólar ($).
Los caracteres siguientes pueden ser letras, dígitos, guiones bajos o signos de dólar.


No se permiten números como primer carácter. De esta forma, JavaScript puede distinguir fácilmente los identificadores de los números.

JavaScript distingue entre mayúsculas y minúsculas

Todos los identificadores de JavaScript distinguen entre mayúsculas y minúsculas.
Las variables lastName y lastname, son dos variables diferentes:

En el siguiente ejemplo, intente cambiar lastName por lastname.

Ejemplo

var lastname, lastName;
lastName = "Moreno";
lastname = "Villanueva";
document.getElementById("demo-11").innerHTML = lastName;

JavaScript no interpreta VAR o Var como la palabra clave var.

Caja JavaScript y Camel

Históricamente, los programadores han utilizado diferentes formas de unir varias palabras en un nombre de variable:

Guiones:
first-name, last-name, master-card, inter-city.
Los guiones no están permitidos en JavaScript. Están reservados para restas.

Guion bajo :
first_name, last_name, master_card, inter_city.

Caso de camello superior (caso de Pascal):
FirstName, LastName, MasterCard, InterCity.

Caso de camello inferior:
Los programadores de JavaScript ofrecen usar mayúsculas y minúsculas que comienzan con una letra minúscula:
firstName, lastName, masterCard, interCity.

Conjunto de caracteres

Unicode cubre (casi) todos los caracteres, signos de puntuación y símbolos del mundo.
