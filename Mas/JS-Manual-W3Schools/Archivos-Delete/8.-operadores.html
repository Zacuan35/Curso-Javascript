Operadores de JavaScript

Asigne valores a las variables y agréguelas juntas:

var x = 5;    // asignar el valor 5 a x
var y = 2;    // asignar el valor 2 a y
var z = x + y;    // asignar el valor de 7 a z (x + y)

x = 5, y = 2, calcular z = x + y, y mostrar z:

<p id="demo"></p>

<script>
    var x = 5;
    var y = 2;
    var z = x + y;
    document.getElementById("demo").innerHTML = z;
</script>

Operadores de cadenas de JavaScript

El operador + también se puede usar para agregar (concatenar) cadenas.

txt1 = "John";
txt2 = "Doe";
txt3 = txt1 + " " + txt2;

El resultado de txt3 será:
John Doe 		<p id="demo"></p>

<script>
    var txt1 = "John";
    var txt2 = "Doe";
    document.getElementById("demo").innerHTML = txt1 + " " + txt2;
</script>

El operador de asignación += también se puede usar para agregar (concatenar) cadenas:

txt1 = "Que día ";
txt1 += "tan bueno";

El resultado de txt1 será:
Que día tan bueno 		<p id="demo"></p>

<script>
    txt1 = "Que día";
    txt1 += "tan bueno";
    document.getElementById("demo").innerHTML = txt1;
</script>

Cuando se usa en cadenas, el operador + se llama el operador de concatenación.

Agregar cadenas y números

Al agregar dos números, se devolverá la suma, pero agregar un número y una cadena devolverá una cadena:

x = 5 + 5;
y = "5" + 5;
z = "Hola" + 5;

El resultado de x, y, y z será:

10
55
Hello5 		<p id="demo"></p>

<script>
    var x = 5 + 5;
    var y = "5" + 5;
    var z = "Hello" + 5;
    document.getElementById("demo").innerHTML =
    x + "<br>" + y + "<br>" + z;
</script>

Si agrega un número y una cadena, ¡el resultado será una cadena!

Operadores Bitwise de JavaScript

Los operadores de bits trabajan en números de 32 bits.
Cualquier operando numérico en la operación se convierte en un número de 32 bits. El resultado se convierte de nuevo a un número de JavaScript.

Operador Descripción Ejemplo Igual que Resultado Decimal
& AND 5 & 1 0101 & 0001 0001 1
| OR 5 | 1 0101 | 0001 0101 5
~ NOT ~ 5 ~0101 1010 10
^ XOR 5 ^ 1 0101 ^ 0001 0100 4
<< Zero fill left shift 5 << 1 0101 << 1 1010 10
>> Signed right shift 5 >> 1 0101 >> 1 0010 2
>>> Zero fill right shift 5 >>> 1 0101 >>> 1 0010 2

Los ejemplos anteriores usan ejemplos de 4 bits sin firmar. Pero JavaScript usa números de 32 bits firmados. Debido a esto, en JavaScript, ~ 5 no devolverá 10. Devolverá -6. ~00000000000000000000000000000101 will return 11111111111111111111111111111010

Bitwise operators se describen completamente en el capítulo JS Bitwise.
