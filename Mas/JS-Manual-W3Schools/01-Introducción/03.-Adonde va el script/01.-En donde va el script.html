¿En dónde va colocado el script?

A considerar:

  • En HTML, el código Javascript se inserta entre las etiquetas <script> y </script>.
  • Puede colocar cualquier cantidad de scripts en un documento HTML.
  • El atributo de tipo no es obligatorio <script type="text/javascript">, JavaScript es el lenguaje de scripting predeterminado en HTML

Javascript puede ir:

  • En el head
  • En el body
  • En ambas partes
  • Como enlace externo
