Creando el formulario

  • Creando el formulario
  • El form tiene un action que se utiliza para enviar los datos, normalmente dentro de las comillas van scripts como "recibe.php" o algún archivo similar, este archivo va a ser el encargado de recibir los datos, procesarlos y hacer algo con ellos como subirlos a una base de datos por ejemplo.
  • Después se encuentra el atributo name para ponerle un identificador al formulario.
  • El método get junto con post son utilizados para enviar datos por el sitio web.
  • Resto de inputs
  • type="text" especifica el input para entrada de texto
  • El maxlenght="30" nos permite decirle que el usuario sólo va a poder escribir 30 carácteres.
  • Después tenemos los input de radio buton
  • Tenemos un input de checkbox
  • Por último tenemos un botón con el type "submit" el cúal nos va a permitir que cuando el usuario presione el botón se envíe el formulario, hay una diferencia muy grande entre submit y simplemente ponerle button, los dos son un botón pero submit te va a enviar los datos específicamente.

Maneras de acceder a form

En estas explicaciones accederemos desde la consola.

Primera forma

  • Accediendo a los formularios
    Escribimos en la consola: document.forms + enter
    Aparecerá lo siguiente HTMLCollection [<form>]
    El código es una colleción HTML, significa que si tenemos mas de un formulario podemos acceder a sus elementos como en un array.
  • Accediendo al primer formulario
    Escribimos document.forms[0] + enter, aparece lo siguiente:
    <form method="get" name="formulario" action="">
  • Accediendo a los inputs
    Después accedemos a los elementos con el atributo elements
    document.forms[0].elements + enter, nos arrojara lo siguiente:
    HTMLFormCollection{0: <input#nombre>, 1: <input#hombre>, 2: <input#mujer>, 3:<input#terminos>}
  • Accediendo a los atributos
    Podemos acceder al primer atributo que sería type="text" escribiendo:
    document.forms[0].elements[0]+ enter
    <input type="text" required="" placeholder="Nombre:" maxlength="30" id="nombre" name="nombre">

Segunda forma

En esta forma se utiliza el atributo name, esta es una forma más fácil de acceder, por que de la otra forma no sabemos si los elementos cambian de posición o el formulario también cambia.

  • La etiqueta form tiene el atributo name de formulario, por ello podriamos acceder mas directo a él escribiendo
    document.formulario + enter, lo que nos arrojará como resultado:
    <form method="get" name="formulario" action="">
  • De igual forma podemos acceder a los elementos hijos escribiendo. document.formulario.nombre + enter y obtendriamos:
  • <input type="text" required="" placeholder="Nombre:" maxlength="30" id="nombre" name="nombre">
  • Y estamos accediendo a lo que es el input de tipo texto pero que tiene el name de nombre.

Propiedades de los formularios

Propiedad name

Nos sirve para conocer el atributo name y acceder directamente a ese elemento, hay dos maneras de escribirlo.
document.formulario.nombre o document.formulario.name
Accediendo al input con name=nombre
document.formulario.nombre+enter
<input type="text" required="" placeholder="Nombre:" maxlength="30" id="nombre" name="nombre">
Conociendo el valor del atributo type
Ya que accedimos escribimos document.formulario.nombre.type+enter
Y como puedes ver nos dice que es de tipo texto.
Esto nos permite hacer muchas cosas con las validaciones, ya que podemos hacer condicionales como recorrer los elementos y a los que sean de tipo texto especificarles cierta situación.

Propiedad value

Podemos conocer el valor del input de texto si tiene un dato ingresado, si escribimos algo de texto en el input de texto aparecerá el valor del contenido al momento de ejecutar la instrucción, por que si después cambiamos el texto tendríamos que volver a ejecutar la instrucción para ver el nuevo valor.
document.formulario.nombre.value + enter

Propiedad focus

Focus sirve para que en el input de texto, el cursor se encuentre por default en el campo de texto y asi facilitar el agregado de un valor.
document.formulario.nombre.focus()
