Modificando el estilo de los elementos

En este capítulo repasaremos la forma de modificar elementos mediante .className y aprenderemos una nueva forma de modificar estilos dinámicamente con Javascript.
Tenemos un encabezado con un id llamado encabezado, una clase que todavía no tiene nada, un contenido y unos estilos agregados al inicio de la página.

Soy un encabezado

Cambiando atributos mendiante .className

Para recapitular lo visto en el capítulo anterior se modificará los elementos del DOM utilizando clases.

  • Agregamos una variable llamada encabezado.
  • La variable será igual a document para acceder al DOM
  • Con .getElementByID(...); accedemos al #encabezado y lo guardarmos en la variable encabezado, para acceder a él más fácilmente.
  • .getElementById("");
var encabezado = document.getElementByID("Encabezado");

Agregando la clase .titulo

Accedemos al encabezado desde consola o el código, y escribimos el código de debajo, y damos enter o actualizamos.
Si vemos su estilo ha cambiado, y si revisamos en el inspector la clase se ha modificado.

encabezado.className = "titulo"

Borrando la clase .titulo

Si queremos borrar la clase titulo basta con reemplazarla con titulogrande, y si queremos conservar ambos solo hay que agregar un espacio y poner la segunda clase.

encabezado.className = "titulo tituloGrande" + enter

Otra forma de eliminar una clase.

  • Accedemos al encabezado.
  • Con className accedemos a la clase.
  • Va a ser igual a encablezado.className o sea si mismo.
  • Vamos a utilizar un método llamado .replace y recibe dos parámetros.
  • El primero es el valor a encontrar y el segundo por cual se reemplazará.
  • Como primer parámetro ponemos "titulogrande"
  • Como segundo parámetro ponemos un espacio en blanco.
  • Actualizamos y vemos el cambio, en la consola revisamos y ya no esta en la clase titulogrande.

encabezado.className = encabezado.className.replace("titulogrande", "");

Cambiando atributos mendiante .style

Es utilizada para acceder y modificar los valores de un atributo style con el cual se puede cambiar cualquier propiedad css, y se puede usar varias veces para cambiar las propiedades de una etiqueta.
Después de escribir los códigos de abajo en el script o en la consola guardamos y actualizamos.

  • Escribimos encabezado seguido de .style con lo cual estamos accediento a los estilos del titular.
  • Despues de un punto accedemos a la propiedad que necesitemos modificar.
  • Despues de un igual escribimos el valor que necesitemos modificar.
encabezado.style.background = "navy";
encabezado.style.color = "#fff";

Propiedades CSS de dos palabras

Para propiedades CSS conformadas por dos palabras, se tiene que seguir la "came case sintax" guardamos y actualizamos.

encabezado.style.paddingTop = "20px";
