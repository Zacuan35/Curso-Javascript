ARRAY - ARREGLO

Los arreglos son variables que permiten almacenar múltiples valores, para acceder a ellos únicamente se le pone el número de la posición de ese valor.
Los arreglos a diferencia de otros lenguajes de programación no tienen que estar especificados en cuanto al número de espacios que tendrán.

Ejemplo;
var amigos = ["Carlos", "César", "Alex"];
document.write(amigos[2]);

Modificando un valor del arreglo

Que pasa si queremos modificar un valor por otro, como el nombre de Carlos por otro nombre, normalmente no se puede cambiar un valor directamente en el array, ya que este se encuentran en ejecución, para ello se tendría que hacer lo siguiente:

Ejemplo;
var amigos = ["Carlos", "César", "Alex"];
amigos[0] = "Arturo";
En este ejemplo se cambia el nombre de Carlos por el de Arturo.

Borrar un valor provisionalmente

Ejemplo:
var amigos = ["Carlos", "César", "Alex"];
amigos[0] = "";

Agregar otro valor al array

Ejemplo:
var amigos = ["Carlos", "César", "Alex"];
amigos[3] = "Manuel";

El método leng nos sirve para saber cuantos elementos tiene un array
