TIPOS DE DATOS / VARIABLES

Podemos almacenar diferentes tipos de información en las variables dependiento de lo que necesitemos y son las siguientes:

  • 1.-Numérico
  • 2.-String - Cadena de texto
  • 3.-Array - Arreglo
  • 4.-Object - Objeto
  • 5.-Boolean - Boleano
  • 6.-Undefined - Indefinido
  • 7.-Null - Nulo
  • 8.-NaN - Error
    •

    Esto nos sirve para saber que es lo que podemos almacenar en una variable y poder decidir con que vamos a trabajar por que muchas veces pensamos que un problema tiene una solución pero realmente tiene otra que es mucho más fácil, los tipos de datos son importantes por que te permiten saber que puedes almacenar para después trabajar con esos valores

    1.-Numérico

    Existen dos tipos de datos numéricos, los entire (entero) y los float (decimal), para los números existen funciones y se pueden hacer muchas cosas como operaciones. En javascript no se tiene que especificar exactamente que tipo de valor se esta trabajando, en otros lenguajes de programación si es necesario.

    1.1 Entire:
    var edad = 19;

    1.2 Float:
    var edad = 19.123;

    2.String - Cadena de texto

    Una cadena de texto puede ser un valor de tipo texto o números, aunque este último será reconocido como un valor numérico por los navegadores, las cadenas de texto van dentro de comillas "...";
    Con las cadenas de texto se pueden hacer varias cosas como extraer un carácter, cortar la cadena de texto y muchas más.

    Ejemplo de la cadena de texto:
    var nombre = "Carlos";
    var numero = "12";

    Una cadena de texto se puede concatenar:
    var nombre = "Carlos";
    var apellido = "Esparza";
    document.write(nombre + " " + apellido);

    Comillas entre comillas.
    No se puede poner comillas dentro de comillas, para ello existen dos opciones:

    1ra opción:
    Reemplazarlas por comillas simples:
    var texto = 'Los mejores videos son de "FalconMasters"';

    2da opción:
    Ponerles una diagonal con comillas inglesas (Aunque tambien funcionan con comillas simples.)
    var texto = "Los mejores videos son de \"FalconMasters\" ";
    document.write(texto);

    3.-Array - Arreglo

    Un arreglo es una variable que almacena un grupo de valores, los arreglos se declaran dentro de brackets [...]; y se puede acceder a ellos poniendo el nombre del arreglo seguido de unos brackets y en su interior el número del elemento al cual queremos acceder, sin embargo en programación los arreglos empiezan desde cero.

    var amigos = ["Carlos", "Alejandro", "César",];
    document.write(amigos[0]);

    Los arreglos nos permiten tener múltiples valores y no necesariamente tienen que ser del mismo tipo.

    var amigos = ["Carlos", "Alejandro", "César", 19, true];
    document.write(amigos[0]);

    4.-Object - Objeto

    Los valores dentro de los objetos se declaran dentro de llaves {...}; es similar a los arreglos ya que también almacenan grupos de valores con la diferencia de que nos permiten llevar un orden.

    var objeto = {
    color: "Black",
    size:12
    }

    5.-Boolean - Boleanos

    Únicamente almacenan dos valores; true y false, y nos sirven tambien para hacer comparaciones.

    var boleano = true;
    var boleano = false;

    6.-Undefined - Indefinido

    Son variables a las que no se les ha definido algún valor, por ello se encuentran en estado indefinido, en el ejemplo, si el código se ejecutara aparecería undefined.

    var falconmasters;
    document.write(falconmasters);

    Desde la consola se puede acceder a las variables.

    7.-Null - Nulo

    Sirve para resetear o modificar el valor de una variable a nulo, a cero.

    var numero2 = 10;
    var numero2 = null;

    8.-NaN - Error

    NaN aparece cuando se quiere realizar una acción u operación ilógica, en el ejemplo se intenta multiplicar un texto.

    var texto2 = "¿Hola cómo estas?";
    document.write(texto * 2);
