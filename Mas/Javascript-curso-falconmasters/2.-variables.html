VARIABLES

Las variables permiten almacenar valores y recurrir a ellas cuantas veces queramos.
A su vez los valores se muestran de manera temporal y son utilizados para visualizarse en pantalla, hacer cálculos u otro tip de tareas.
Los valores de texto se agregan entre comillas.
Existen varios tipos de variables y de datos.
Las variables inician con la palabra var, seguidas de un nombre al que se le asigna un valor, aunque si no llevaran la palabra var e iniciaran con el nombre, los navegadores tambien lo entenderían. En las variables podemos agregar números

Ejemplo:
var Nombredelavariable = "Valordelavariable";
alert(Nombredelavariable);

En la primera línea del código tenemos una variable con el nombre "Nombredelavariable", la cual almacena un valor que es un texto.
En la segunda línea de código hay una función de alerta con la variable "Nombredelavariable", lo que significa que se va a mostrar el valor que tenga esa variable.

Función prompt

Una función también es un valor, la mayoría del tiempo siempre va a ser un dato o un valor que se puede almacenar, es por eso que en javascript las funciones tambien muchas veces se van almacenar en una variable.
La función promp muestra un cuadro con un campo para ingresar datos.

Ejemplo:
var nombre;
nombre = prompt("Ingresa tu nombre");
alert(nombre);

document.write

Este método sirve para mostrar datos en pantalla, escribiendo en el documento .html el valor que tenga contenido dentro.

Ejemplo:
var nombre;
nombre = prompt("Ingresa tu nombre");
document.write(nombre);

En el ejemplo la primera línea sirve para declarar la variable que se llama "nombre".
La segunda línea indica que la variable nombre es igual al el valor dentro de la función prompt.
Por último en la tercera línea la función document.write esribirá en el documento .html el valor de la variable nombre.

console.log

El valor contenido dentro de console.log se verá reflejado únicamente en la consola.

Ejemplo:
var nombre
nombre = prompt("Ingresa tu nombre");
console.log(nombre);

Variables concatenadas

Dentro de las variables se pueden concatenar valores, en el siguiente ejemplo se tienen dos variables que se llaman nombre y edad, como valores tienen funciones de promp, por último la línea de document.write concatena los valores de las dos variables y las muestra en pantalla.

Ejemplo:
var nombre;
nombre = prompt("Ingresa tu nombre");
edad = prompt("Ingresa la edad");
document.write(nombre + " " + edad);

Operaciones dentro de las variables

Dentro de las variables se pueden hacer operaciones como las siguientes:

Ejemplo:
nombre = "Carlos";
edad = 19 + 1;
document.write(nombre + " " + edad);

nombre = "Carlos";
edad = 19 * 2;
document.write(nombre + " " + edad);
