ARRAY - MÉTODOS Y PROPIEDADES

Estos son algunos de los métodos y propiedades más utilizados para las variables array:

Propiedad .length

Cuenta la cantidad de valores contenidas en un arreglo. Para usarlo basta con poner el nombre del arreglo seguida de .length

En el siguiente ejemplo el valor que se nos mostrará en pantalla será 3, por que es el número de valores dentro del array amigos.
var amigos = ["Carlos", "César", "Alex"];
document.write( amigos.length );

Ahora se le dará un poco mas de forma al mensaje para que el texto mostrado en pantalla sea: tienes 4 amigos

var amigos = ["Carlos", "César", "Alex", "Manuel"];
document.write("Tienes" + amigos.length + "amigos");

Agregando un nuevo valor con .length

Ejemplo
var amigos = ["Carlos", "César", "Alex", "Manuel"];
amigos [amigos.length] = "Fernando";
amigos [amigos.length] = "Luis";

1.-En la primera línea se tiene un array llamado amigos con 4 valores.
2.-En la segunda línea se indica que se accederá a un valor del array llamado amigos, dentro de los brackets en lugar de escribir un número se pone la variable amigos seguida de la propiedad.length, esto significa que dentro de los brackets el valor existente es 4, por que la propiedad estará contando los valores del array amigos.
Entonces como en programación se empiezan a contar desde 0, el numero de elemntos estará un numero arriba que lo contado por el navegador, es decir 4, y 4 = a "Fernando", y como no hay una posición 4 en el array Fernando se coloca hasta el final del arreglo.

Propiedad .push ("...");

Es la propiedad correcta para agregar valores directamente al final de un arreglo sin tener que hacer lo anterior.
Para usarlo basta con poner el nombre del arreglo seguida de .push ("valor a agregar");

Ejemplo
var amigos = ["Carlos", "César", "Alex", "Manuel"];
amigos.push("Luis", "Fernando", "Oscar");

Propiedad .pop ( );

Con la propiedad .pop se elimina el último valor de un array, para utilizarlo basta con poner el nombre del array seguido de .pop ();

Ejemplo
var amigos = ["Carlos", "César", "Alex", "Manuel"];
amigos.pop();
document.write(amigos[0]);
document.write(amigos[1]);
document.write(amigos[2]);
document.write(amigos[3]);

.concat ( "..." );

.concat sirve para juntar los valores de varios arreglos, para usarlo basta con poner el nombre del array seguido de .concat (); y agregar en el interior de los paréntesis el array a sumar.

Ejemplo
var amigos = ["Carlos", "César", "Alex", "Manuel"];
var amigos2 = ["Luis", "Fernando", "Óscar"];
var amigos 3 = amigos.concat(amigos2);
document.write(amigos3);

Método .join ( );

Este método permite dividir los valores de un array con lo que nosotros le asignemos como un espacio, letra, número etc.
Para utilizarlo basta con poner el nombre del array seguido de .join ("espacio, carácter, letra o número a asignar");

Ejemplo
var amigos = ["Carlos", "César", "Alex", "Manuel"];
amigos.join(" , ");

Para verlo escrito en pantalla:
var amigos = ["Carlos", "César", "Alex", "Manuel"];
docuemnt.write( amigos.join(" , ") );

Una forma más correcta de escribir lo anterior:
var amigos = ["Carlos", "César", "Alex", "Manuel"];
var amigosconseparacion = amigos.join( , );
document.write( amigoscon separacion );

.reverse ( );

.reverse ordena los elementos de un array, para utilizarlo basta con poner el nombre del array seguido de .reverse ();

Ejemplo
var amigos = ["Carlos", "César", "Alex", "Manuel"];
var amigosordenados = amigos.reverse();
document.write(amigosordenados);
