Creando nodos del DOM

En este capítulo vamos a estar aprendiendo como crear nodos y como agregarlos a nuestro código HTML

Estos son los pasos para poder agregar un elemento o nodo al documento:

  1. Crear el elemento
  2. Crear un nodo de texto
  3. Añadir el nodo de texto al elemento
  4. Agregar atributos al elemento
  5. Agregar el elemento al documento

1.- Creando el elemento
Crearemos una variable llamada elemento y será igual a un document seguido del método .createElement el cual nos permite crear un elemento escribiéndolo dentro de los paréntesis.

var elemento = documet.createElement("h2");

2.- Creando el contenido
Escribiremos otra variable llamada contenido, y será igual a document seguido del método .createTextNode, y dentro de los paréntesis pondremos todo el texto de nuestro contenido.

var contenido = documet.createTextNode("Este es nuestro titular");

3.- Añadiendo el contenido al elemento
Hay que añadir el contenido al elemento para juntarlos, para ello se llama a la variable elemento seguida del método .appenChild("..."); dentro de los paréntesis recibirá como parámetro el contenido que añadiremos.

elemento.appendChild(contenido);

4.- Agregando atributos al elemento (Opcional)
Accedemos a elemento seguido del atributo .setAttribute y dentro de los paréntesis recibirá dos valores, el primero es el nombre de la propiedad y el segundo el valor.

elemento.setAttribute("align", "center");

5.- Agregando el elemento al documento
Accederemos a document seguido de .getElementById y dentro de los paréntesis escribimos el id subtitulo, seguidamente escribimos el método .appendChild y dentro de los paréntesis vamos a agregar el elemento que hemos creado, o sea el h2 con su contenido.
Lo que hemos hecho fue crear un nodo y agregarlo al sitio.

document.getElementById("subtitulo").appendChild(elemento;)

Para agregar elementos al final del body sería de la siguiente forma:

document.body.appendChild(elemento;)
