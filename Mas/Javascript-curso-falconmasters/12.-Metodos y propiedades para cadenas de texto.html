Métodos y propiedades para cadenas de texto

Propiedad .length;

.length al igual que en los arreglos, funciona para saber cuantos carácteres componen una cadena de texto.
En el siguiente ejercicio el script muestra el número 13 en pantalla.
Los espacios también cuentan como un carácter.

var texto = "Carlos Arturo";
var nuevoTexto = texto.length;
document.write(nuevotexto);

Propiedad .substring (...);

Substring nos permite cortar una cadena de texto, por ejemplo si le ponemos el número 3 y actualizamos, nos corta la cadena de texto desde la posición 3 y hasta el final.

var texto = "Carlos Arturo";
var nuevoTexto = texto.substring(3);
document.write(nuevotexto);

Propiedad .substr (...);

Con .substr podemos pasar dos parámetros, el primero sería el índice de inicio, y el segundo el número de carácteres a extraer.
En el ejemplo se indica como inicio de corte la posición 0, y corta 6 carácteres.

var texto = "Carlos Arturo";
var nuevoTexto = texto.substr(0, 6);
document.write(nuevotexto);

Propiedad .indexOf("...");

indexOf nos devuelve la posición de una letra en específico.
En el ejemplo la propiedad nos dirá en que posición se encuentra la letra "a".

var texto = "Carlos Arturo";
var nuevoTexto = texto.indexOf("a");
document.write(nuevotexto);

indexOf es sensible a minúsculas y mayúsculas, por lo cual no le representa confusión a la hora de buscar la letra "a", pero que pasa si escribimos que busque la letra "o", como en este caso hay dos minúsculas devolvera como resultado sólo la posición de la primera letra o que encuentre, que será 4.

var texto = "Carlos Arturo";
var nuevoTexto = texto.indexOf("o");
document.write(nuevotexto);

Nosotros podemos indicarle desde donde queremos que empiece a buscar, esto lo podemos hacer pasando un segundo parámetro.
En el ejercicio se indica que desde la posición 5 empezará a buscar las letras "o" y nos da como resultado 12.

var texto = "Carlos Arturo";
var nuevoTexto = texto.indexOf("o", 5);
document.write(nuevotexto);

Si no le queremos poner el número por que muchas veces lo desconoceremos, podemos hacer lo siguiente:

var texto = "Carlos Arturo";
var texto2 = texto.indexOf("o") + 1;
var nuevoTexto = texto.indexOf("o", texto2);
document.write(nuevotexto);

Propiedad lastIndexOf

Hace algo similar sólo que empieza a buscar desde atrás hacia adelante.
En el ejercicio al buscar la letra "o" nos devuelve como resultado el número 12.

var texto = "Carlos Arturo";
var nuevoTexto = texto.lastIndexOf("o");
document.write(nuevotexto);

Propiedad .replace (...);

.replace nos permite reemplazar una cadena de texto por otra.
Este método recibe dos parámetros, el primero es la variable a modificar, y el segundo el texto a reemplazar.
En el ejemplo queremos cambiar el texto de la variable "texto" por FalconMasters.

var texto = "Carlos Arturo";
var nuevoTexto = texto.replace(texto, "FalconMasters");
document.write(nuevotexto);

Incluso podemos pasar otra cadena de texto con el texto que quieramos.
En el ejemplo hemos sustituido la palabra Carlos por falconMasters

var texto = "Carlos Arturo";
var nuevoTexto = texto.replace("Carlos", "FalconMasters");
document.write(nuevotexto);

También podemos reemplazar letras por signos.

var texto = "Carlos Arturo";
var nuevoTexto = texto.replace("a", "-");
document.write(nuevotexto);

toUpperCase ();

Nos permite convertir nuestro texto a mayúsculas.

var texto = "Carlos Arturo";
var nuevoTexto = texto.toUpperCase();
document.write(nuevotexto);

toLowerCase();

Nos permite convertir nuestro texto a minúsculas.

var texto = "Carlos Arturo";
var nuevoTexto = texto.toLowerCase();
document.write(nuevotexto);
