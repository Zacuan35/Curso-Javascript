Ejercicio con funciones y formularios

En este capítilo estaremos viendo una pequeña calculadora para que te des una idea de como poder trabajar con funciones, y también introducir un poquito lo que son los formularios.
Se realizará un formulario con dos input para agregar datos, y que al darle click al botón sumar nos aparezca el resultado.

1.-Creando el formulario html

Se va a agregar dos input de tipo texto con indicadores "id" los cuales se llamarán numero1 y numero2.
Después vamos a agregar otro input de tipo "button", va a tener un "value" que diga sumar.

2.- Creando la función que sume los números

Esto se realiza dentro del script de la siguiente manera:

2.1.-Una variable llamada suma que es igual a una función, esta función recibe dos parámetros, numero1 y numero2.
2.2.-Dentro se crean dos variables llamadas numero1 y numero2 que será igual a los id llamados numero1 y dos.
2.3.-Los id numero1 y numero2 se va a contener dentro de "parseFloat", la cual es una propiedad que nos ayuda a convertir un dato de tipo "cadena de texto" a número decimal, sin él la operación no se realizaría.
La función parseFloat recibe como parámetro el document.getElementById y lo tranforma en decimales, también podría funcionar con la función parseInt pero este solamente transforma enteros y no toma en cuenta decimales.
Hay otras formas de hacerlo para que aparezcan sólo dos decimales.
2.4.-Dentro de parseFloat a modo de parámetro sucede lo siguiente:
Se obtiene un elemento del documento .html y se extraé su valor, lo anterior se desglosa de la siguiente manera:
Del documento .html ".document" se busca ".getElementById" el elemento que tenga el Id "numero1 y numero2" y se extraé su valor ".value".
2.5.-Se crea una variable llamada resultado que es igual a la suma de numero1 + numero2.
2.6.-Por último con "return resultado" se regresa la variable llamada resultado.

var suma = function(numero1, numero2){
  var numero1 = parseFloat(document.getElementById("numero1").value);
  var numero2 = parseFloat(document.getElementById("numero2").value);

  var resultado = numero1 + numero2;
  return resultado;
}

3.-Agregando un evento

Para que aparezca el resultado de la operación al darle click al botón, se tiene que agregar un evento.

3.1.- Dentro del input del botón se agrega el atributo "onclick"
3.2.- Como valor de "onclick" se coloca un "alert ( )", ya que se quiere mostrar algo en pantalla.
3.3.- Dentro del valor "alert ( )" se pone suma y sus parámetros.

Los puntos anteriores se resumen en lo siguiente;
Al darle click (onclick) al input (button), va a ejecutar un (alert) que tenga el resultado de la función llamada suma.

La forma correcta de agregarlo es dentro de un archivo externo .js, pero para efectos del ejercicio se agregará dentro del código .html
