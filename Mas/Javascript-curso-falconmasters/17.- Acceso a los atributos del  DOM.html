Accediendo a los atributos del DOM

En este capítulo vamos a estar trabajando los atributos de los elementos dentro del DOM, para modificarlos, agregar nuevos etc.
Empezaremos con estos dos Divs.


Creando el elemento

  • Se crea una variable llamada elemento y es igual a un document, con ello se está entrando al contenido de la página web, que es el árbol de DOM.
  • Seguido se escribe el método createElement(...);
  • Dentro de los paréntesis se escibe el elemento a crear.

Accediento al elemento padre

  • Se crea la variable llamada padre que igual al #contenedor.
  • La variable padre es igual a un document
  • Seguido de document se escribe el método .getElementById(...); con el #contenedor.

Estableciendo un punto de referencia

  • Creamos la variable referencia que va a ser igual al primer elemento "li"

Posicionando el nodo al principio .insertBefore (...);

  • Llamaremos a la variable padre seguida de .insertBefore(...);
  • Dentro de los paréntesis se agrega el elemento creado y la referencia.
Otra forma de insertar el elemento creado es con body.appenChild(...); pero lo coloca hasta el final fuera del contenedor
var elemento = document.createElement("div");
document.body.appendChild(elemento);

var elemento = document.createElement("div");
var padre = document.getElementById("contenedor");
var referencia = document.getElementByTagName("div")[0];
padre.insertBefore(elemento, referencia);

Cambiando los atributos del DOM

Existen las siguientes maneras de acceder a los atributos:

  • .setAttribute (attributename, attributevalue);
  • .className="" Se utiliza para acceder a un selector de atributo css.
  • .idSe utiliza para acceder a un selector de atributo de id.
  • Acceder directamente al atributo de un elemento html

.setAttribute (attributename, attributevalue);

  • Se utiliza para agregar o cambiar valores de atributos html como lo son href, src, alt, style etx.
    Si una etiqueta no tiene un atributo declarado este método sirve para insertarlo, pero si ya tiene uno sirve para reemplazarlo.
  • Se utiliza para agregar atributos de clase o id.

Agregando un atributo tipo class

  • En un script o en la consola llamamos a la variable elemento seguido de .setAttribute (...);
  • Dentro de los paréntesis recibe dos parámetros:
    • 1.-El nombre del atributo que se va a agregar: atributo de clase.
    • 2.-El nombre del valor que se va a agregar: azul.
  • elemento.setAttribute("class","azul"); + enter

Agregando un atributo tipo id

  • En un script o en la consola llamamos a la variable elemento seguido de .setAttribute (...);
  • Dentro de los paréntesis agregamos el atributo id con su valor
  • elemento.setAttibute("id", "nuevaID") + enter

El método respeta los espacios y se pueden agregar dos id o class.
elemento.setAttibute("id", "nuevaID otraID") + enter

Accediento por medio de un selector de atributo

.id

En la consola o en el script escribimos lo siguiente:
Y si lo revisamos en el inspector ya se cambió el Id por tercerID.
elemento.id = "tercerID" + enter

.className=""

En la consola o en el script escribimos lo siguiente:
elemento.className = "Rojo" + enter

Accediento directamente al atributo de un elemento HTML

Se a creado una etiqueta de link con el siguiente selector #enlace.

FalconMasters

Desde la consola se accede directamente al atributo de href de una etiqueta de enlace para cambiar su valor.

  1. Accediendo al documento: Se llama al script o directamente en la consola al documento "document"
  2. Accesando a la etiqueta: Se hace mediante el selector Id que se encuentra en la etiqueta .getElementById (enlace) + enter
  3. Accesando al atributo: Con .href + enter se accede directamente al atributo.
  4. Cambiando el enlace: Dentro de las comillas de href se cambia el enlace.
document.getElementById("enlace").href = "http://ww.google.com" + enter
