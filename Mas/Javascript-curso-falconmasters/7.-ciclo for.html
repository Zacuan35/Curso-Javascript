BUCLE FOR

Que pasa si por ejemplo queremos mostrar números en pantalla del uno al diez, clases atras se hubiera hecho lo siguiente:

  document.write("1");
  document.write("2");
  document.write("3");
  document.write("4");
  document.write("5");
  document.write("6");
  document.write("7");
  document.write("8");
  document.write("9");
  document.write("10");

Pero que ocurre si se tienen cien números, esta tarea se haría ineficiente, el modo de trabajar sería incorrecto, por que hay formas mucho más fáciles y esto nos permiten hacer los ciclos.
Los ciclos nos permiten repetir una estructura como document.write ("valor x"); y que en cada ejecución del ciclo, es decir que en cada repetición dicho valor sea el que cambie.
En el ejemplo de los números, la primera ejecución sería uno y la última sería cien.

Los bucles son sentencias de interacción que ofrecen una manera rápida de hacer algo repetidamente.

Hay muchas clases diferentes de bucles, pero todos ellos hacen en esencia lo mismo: repetir una acción un numero de veces. Los diferentes bucles ofrecen también diferentes formas de determinar sus puntos de inicio y final. Hay varias situaciones que son mas fácilmente resueltas por un tipo de bucle que por otros.

Un bucle for se repite hasta que la condición especificada sea evaluafa como false, y se utiliza cuando cuanto se conoce el número de veces que se ejecutará un código, pero no siempre será asi.

Sintáxis

Se escribe for seguido de paréntesis y llaves:

for( ) {
  //Instrucciones
}

Paréntesis: Dentro de los paréntesis van tres indicadores de la variable.
1.-Expresión inicial
2.-Condición
3.-Expresión de incremento

Llaves: Dentro de las llaves irán las instrucciones que se ejecutarán si los indicadores del bucle se cumplen.

for( 1.-Expresión inicial; 2.-condición; 3.-Expresión incremento) {
  //Instrucciones
}

1.-Expresión inicial.

1.- Determina un punto de inicio que casi siempre va a ser 0, para ello basta con poner el nombre de la variable seguida de un signo igual.
En este ejemplo la variable se llamará "i", ya que es como se le va a encontrar la mayoría de las veces, pero puede nombrarse de otra manera. La variable puede iniciar con la palabra reservada var, pero puede ir sin ella, no hay ningún problema.

la expresión inicial permite una expresión con cualquier grado de complejidad, pudiendo contener funciones que nos permiten reutilizar código.

Ejemplo:
for( i=0; ){
  //Instrucciones
}
El valor de la variable "i" será de "0" desde la primera ejecición.

2.- Condición.

El valor de la variable se repetirá "HASTA QUE" la condición sea evaluada como verdadera.
En el ejemplo mientras que la variable i sea menor o igual a diez, el valor se repetirá.

for( i=0; i<=10; ){
  // Se ejecuta 9 veces, del rango 0 al 9.
}

3.- Expresión de incremento.

Especifica la acción que ocurrirá por cada ciclo ejecución, lo que la mayoría del tiempo se va a a encontrar es que la variable va a incrementar (i++) o o decrecer (i--).

for( i=0; i<=10; i++ ){
  //Instrucciones
}
En cada ejecución del ciclo la variable "i" incrementará en una unidad.

Ya por último se ponen las instrucciones contenidas en las llaves.

for( i=0; i<=10; i++ ){
  document.write( "Ou yeah!" + "<br>");
}

Analizando las instrucciones de las llaves

Como se puede observar el texto Ou yeah! se repitio once veces, por que empezamos a contar desde el 0, y del cero al 10 hay once números, para corregirlo podemos cambiar el 0 por 1.

for( i=1; i<=10; i++ ){
  document.write( "Ou yeah!" + "<br>");
}

Ejecuntando la unidad "i"

En el document.write la unidad "i" empieza de 0 y a partir de ahí comoeinza a correr el valor.

for( i=0; i<=10; i++ ){
  document.write( i + "<br>");
}

Ejemplo adicional, agregando variable que equivale a un número.

var numeroDeUsuarios = 50;

for( i=0; i<=numeroDeUsuarios; i++ ){
  document.write( i + "<br>"); }

"i++ es igual a poner i = i + 1"

Utilizando .length

Lo que aparece en el código es lo siguiente:
1.-Hay un arreglo llamado días que tiene 7 valores de tipo texto.
2.-Sin embargo suponiendo que desconocemos el dato del número de valores utilizamos la propiedad .leng, la cual nos va arrojar el número de valores que son 7.
3.-Utilizando el ciclo for se establece que la variable i comenzará desde 0, es decir desde el primer valor (Lunes), "hasta que" sea menor o igual a dias.length (7), y que en cada ciclo este se sumará un valor i.
Si todo lo anterior resulta como verdadero se ejecutará lo siguiente:
Se escibirá en pantalla, la unidad i del arreglo días, como i es igual a 0, lo que extraerá de días será el valor "Lúnes", dato que esta concatenado a un salto de línea.

var dias = [ "Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];

dias.length = 7
for( i=0; i<=dias.length; i++ ){
  document.write( dias[i] + "<br>");
}

En el ejemplo anterior; se extrae la unidad "i" de dias, como la unidad "i" empieza de 0 extrae el Lúnes y con ese texto empieza el cilo.

Arreglando undefined

Debido a que en programación se inicia desde el 0, el valor de 7 de .length hace que tengamos un valor final inexistente, por lo cual se le restará un número para que el ciclo se repita hasta que sea igual o menor a 6.

var dias = [ "Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];

dias.length = 7
for( i=0; i<=dias.length - 1; i++ ){
  document.write( dias[i] + "<br>");
}

Para saber el número de valores del arreglo días utilizamos .length

Al parecer i es considerado por javascript como una unidad, como un valor que por default significa uno aunque no este declarado en ninguna variable.
