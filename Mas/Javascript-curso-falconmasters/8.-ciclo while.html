BUCLE WHILE

While se usa cuenado se desconoce el límite de veces que se ejecutará una acción, por ejemplo decir: camina hasta encontrarse con un letrero amarillo, o corre hasta llegar a la meta.

Sintaxis

Se escribe la variable con su valor correspondiente, después la palabra while, seguida de la condición entre paréntesis, y entre llaves la expresión que se repetirá mientras que la condición sea verdadera, seguida de la expresión de incremento i++;.
El ciclo while también lleva tres indicadores, pero como puedes observar no van juntas dentro de los paréntesis sino uno antes de la declaración while, otro en los paréntesis, y el último al final de la expresión.

var i=0;
while (condition){
  expression i++
}

La variable i puede ser diferente, depende de la lógica del programa.
La expresión de incremento esta detro de las llaves, y el valor de la variable "i" cambiará si la condición resulta verdadera.
A diferencia del ciclo for, no se puede poner true en los paréntesis para que se ejecute la acción, por que sería infinito, a menos de que se agregue una variable que sea igual a true y ya se podría cambiar a false. y si ponemos false no se ejecutaría nunca.

Condición infinita

Hay que tener en cuenta que no estemos escribiendo una condición infinita, en el ejemplo siguiente si no llevara la expresión de incremento, las instrucciones se ejecutarian infinitamente, por que el valor de i = 0 siempre sería el mismo y nunca llegaría al rango final de la condicional. La condición se debe plantear "mientras que" el valor resultante de esto sea verdadero

var i=0;
while (i <= 10) {
  document.write(i + "<br>");
}

Ejemplo

Volvemos a escribir el arreglo de los días, después se va agregar la variable numeroDias que será igual a dias.lenght, es decir, a la longuitud del arreglo - 1, y la variable "i" se repetirá mientras sea menor o igual a númeroDias.
Entonces, si el valor "i" es igual a numeroDias (6), se ejecutarán las instrucciones y se escribirá en pantalla la unidad de la variable días "i", lo que significa que se extraerá el número 0 de días.


var dias = [ "Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];

var numeroDias = dias.length - 1;

var i = 0
while (i <= numeroDias) {
  document.write(dias[i] + "<br>"); i++;
}

Do while

El ciclo do while primero va a ejecutar unas instrucciones siempre, aunque no se cumpla la condición, la condición la va a preguntar hasta el último, y si es verdadera va a seguir ejecutando el resto del código.
