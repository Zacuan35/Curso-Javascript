En este capítulo vamos a interactuar con el documento, a trávez de un botón transformaremos las dimensiones de una imagen cambiando con javascript las clases que definen sus dimensiones.
Al cargar la página su tamaño será de 200px de ancho y al dar clic al botón cambiará a 400px de ancho.

Creando el botón

Crearemos un formulario para agregar un botón.

  • Crearemos un elemento input de tipo boton: el cuál activará el cambio de tamaño.
  • Insertamos el atributo de valor: Para botones define el texto del botón.
  • Utilizaremos el atributo onclick: al presionar el botón se va a ejecutar la función "zoom();"
  • "Utilizar eventos de esta forma no es lo más correcto"

Insertando la imagen

  • Se inserta una imagen con un ancho de 400px * 400px
  • Aparte le agregaremos un width de 400px
  • Le insertamos una clase llamada thumb que le resetea las dimensiones a 200px.
  • Tambien lleva una propiedad de transición para que el cambio sea suave.
  • Le colocamos un identificador id con el nombre thumb.

Cuando el zoom no este activado el tamaño será de 200px, y con javascript agregaremos y quitaremos la clase .grande.


Creando la función

  • Guardaremos la función dentro de una variable llamada zoom.
  • Dentro de la declaración creamos una variable llamada thumb, con lo cuál accederemos a la etiqueta de imagen.
  • Ahora se llama a la variable "thumb" (Etiqueta imagen) y se reemplaza la clase thumb por la de thumb grande.
    "Se puede hacer con .className y sería el siguiente código:
    thumb.className = "thumb grande";"

Actualizamos la página, esperamos a que cargue, presionamos el botón y la imagen se ha hecho más grande al presionar,

var zoom = function(){
  var thumb = document.getElementById("thumb");
  thumb.setAttribute("class" , "thumb grande");
}

Comprobando si funciona

Para comprobar de momento si funciona podemos poner un alert con cualquier mensaje, actualizamos, presionamos el botón, y si aparece el mensaje significa que todo esta correcto de momento.

var zoom = function(){
  var thumb = document.getElementById("thumb");
  alert("s");
}

Insertando condicionales

Sin embargo queremos que al presionar de nuevo se vuelva a hacer más pequeña, y para ello debemos quitar la clase grande.
Para esto tenemos que usar condicionales.

  • if
    Si la clase que esté dentro del Id thumb es igual a la clase thumb se ejecutará la siguiente declaración:
    La clase dentro del id thumb cambiará a thumb grandre.
  • else
    Pero si la clase que esta dentro de Id thumb no es igual a la clase thumb se ejecutará la siguiente declaración:
    La clase dentro del id thumb cambiará a thumb.

En el inspector checamos que tiene la clase thumb y no la de thumb grande.
Presionamos el botón y se hace más grande, y la clase cambia.
Es importante fijarse en el inspector como al presionar el botón las clases se agregan y se quitan dinámicamente.

var zoom = function(){
  var thumb = document.getElementById("thumb");
  if (thumb.class == "thumb"){
    thumb.className = "thumb grande"
  }else{
    thumb.className = "thumb"
}
}
