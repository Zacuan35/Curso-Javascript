Introducción al DOM (Document object model)

El objetivo del DOM es poder acceder y manipular los elementos de un sitio web con javascript de una manera fácil, como por ejemplo cuanto se accede a formularios para poder obtener un valor y después trabajar con el.
El DOM básicamente es la estructura de una página web pero conformada por elementos llamados nodos, consta de dos elementos primarios que son el head y el body y dentro de cada uno de ellos tenemos más elementos, cada uno de estos elementos son nodos que podemos acceder en el DOM.

¿Cómo podemos acceder a los elementos con el DOM?

Hay varias formas de hacerlo, la primera de ellas es seleccionando todos los elementos de nuestro sitio web por etiquetas o buscándolos mediante un ID.

Creando el script

Mediante la siguiente variable accederemos a los elementos:

1.-Se crea una variable llamada elementosP, para acceder a todos los elementos p que tenemos en pantalla.
2.-Después de la variable escribimos igual a .document por que es la raíz de nuestro DOM.
3.-Enseguida escribimos getElemenstByTagName ("..."); esta sintáxis es conocida como *camelsintax, y dentro de los paréntesis pondríamos la etiqueta que queremos alcanzar, en este caso todos los elementos "p".

var elementosP = document.getElementsByTagName("p");

Camel sintax en español sintáxis del camello, se llama asi por que la primer letra va a ser minúscula siempre, pero la primera letra de cada palabra siguiente va en mayúscula.

1.-Trayendo los elementos mediante etiquetas

Supongamos que queremos obtener la etiqueta número dos del elemento "p".
"En este ejercicio es importante tener la etiqueta script hasta el final del código por que necesita cargar todas las etiquetas para poder trabajar con ellas".

Accediendo a la variable mediante la consola

Vamos a ir a nuestra página web y actualizamos, nos dirigimos a la consola y escribimos elementosP + enter, nos dice que es una colección html y que podemos acceder a ellos como si fueran un arreglo.

Accediendo al primer elemento y modificando su contenido

1.-En la consola escribimos elementosP y unos [ ], ya que vamos a ponerle sintáxis de arreglo.
elementosP[...]
2.-Dentro de los brackets escribimos el número 0 por que estamos accediendo desde la posición 1, damos un enter y nos dice que hemos accedido al primer párrafo, incluso en pantalla nos lo está subrayando, y si le damos click nos enviará al modo inspector y nos seleccionará el primer párrafo.
elementosP[0]
3.-Seguidamente escribiremos .innerHTML y damos un enter, con el cual vamos a acceder a todo el contenido html que este dentro de estas etiquetas, en este caso es texto y etiquetas.
elementosP[0].innerHTML
4.-Ahora modificaremos el elemento 1, después de .innerHTML escribimos que va a ser igual a "ou yeah"; damos un enter y en pantalla a cambiado nuestro elemento.
elementosP[0].innerHTML = "ou yeah";

2.- Trayendo los elementos por su ID

Otra forma de acceder a los elementos y que es la más habitual, es trayendo los elementos por su ID.
Se creará una variable llamada segundoParrafo la cual es igual al documento .document, y va a acceder al el .getElementById("...");, dentro de los paréntesis se escribirá el id del elemento que se quiere buscar, en este caso el segundo parrafo que tiene el id llamado segundo.

var segundoParrafo = document.getElementById("segundo");

1.-Guardamos, actualizamos, y accedemos en la consola al segundo párrafafo escribiendo:
segundoParrafo + enter
nos dirá que tenemos el elemento p con el id"segundo"
2.-Si le agregamos .innerHTML nos aparecerá el contenido:
segundoParrafo.innerHTML
