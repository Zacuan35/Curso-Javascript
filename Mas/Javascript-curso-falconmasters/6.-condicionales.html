CONDICIONALES if - else

Si una condición especificada es evaluada como verdadera ejecutará una acción. Si la condición es evaluada como falsa, otra sentencia puede ser ejecutada.

Sintáxis

Las condiciones comienzan con un if seguido de paréntesis y llaves if (...){...}; dentro de los paréntesis va escrita la condición que va a ser lo que queremos comprobar, y dentro de las llaves van escritas las instrucciones que se ejecutarán si la condición se cumple, y si no tenemos else (...){...}; en el que van escritas otras instrucciones.

if ( condición){
  // instrucciones
} else {
  // instrucciones
}

Utilizando operadores

Los operadores de comparación sirven para definir de una manera más dinámica las condiciones y son las siguientes:

OPERADORES DE COMPARACIÓN
< Menor que
> Mayor que
>= Mayor o igual
<= Menor o igual
== Igual
!= Distinto de

En el siguiente ejemplo si nombre es igual a Carlos, se ejecutará la acción contenida dentro de las llaves.

var nombre = "Carlos";

if( nombre == "Carlos" ){
  document.write("Bienvenido" + nombre);
}

Else

Else puede traducirse como "sino", y sirve para poner una condición adicional en caso de que no se cumplan ni la primera ni la segunda.

var nombre = "Carlos";

if( nombre == "Carlos" ){
  document.write("Bienvenido" + nombre)
} else {
  document.write("Bienvenido anónimo");
}

En el siguiente ejemplo la primera condición no se cumple por lo que se ejecuta la contenida en else:

var nombre = "Carlos",
edad = 15;

if (edad == 18) {
  document.write("Bienvenido, tienes 18 años");
} else {
  document.write("No tienes 18 años");
}

Else if

Else if es una expeción y sirve para poner otra condición en caso de que no se cumpla la primera.

var nombre = "Arturo",
edad = 15;

if( nombre == "Carlos" ){
  document.write("Bienvenido" + nombre)
} else if ( nombre == "Arturo" ) {
  document.write("Bienvenido" + nombre) }

El else del final es por si todo lo demás no se ejecutó

var nombre = "Carlos",

if (edad == 18) {

OPERADORES LÓGICOS
And & & Significa "y" y podemos agregar cuantas necesitemos.
Or || Permite indicar que se puede cumplir cualquiera de las condiciones de la condicional.

Operado lógico "And"

Con el operador lógico "And" dos condiciones son evaluadas por el navegador, si ambas se cumplen se ejecuta la acción.

var nombre = "Arturo",
edad =18;

if ( edad >= 18 && nombre == "Carlos"){
  document.write("Bienvenido, eres mayor de edad");
} else {
  document.write ("No estas permitido");
}

Operador lógico Org ||

El operador Org es similar a And, por que también dos o mas condiciones son evaluadas, pero a diferencia de And, si alguna de las condiciones resulta verdadera se ejecutará la acción.

var nombre = "Arturo",
edad =18;

if ( edad >= 18 || nombre == "Carlos"){
  document.write("Bienvenido, eres mayor de edad");
} else {
  document.write ("No estas permitido");

edad =18;

}

Funciones dentro de condiciones

Las funciones que se encuentran de esta forma retornan valores:

if( funcion ){
}

Para que una condición se ejecute tiene que tener como valor true o false

if( true ){
  document.write("OK");
}

Condicionales para saber si una variable tiene asignado un valor.

Lo mismo pasa con las variables, aqui lo que el navegador evaluará será si hay algún valor asignado a la variable "nombre", si es asi se ejecutará la acción. Muchas veces se van a crear condicionales para preguntar si hay un valor que ha sido agregado o no a una variable.

var nombre = "Arturo",
edad =18;

if( nombre ){
document.write("OK");
}

Tres signos de igual

Tres signos de igual, se utiliza para comprobar que tengan el mismo valor y sean del mismo tipo, en el ejemplo tienen el mismo valor "18", pero son de diferente tipo, ya que mientras un valor es de tipo entero, el otro es de tipo texto.

var nombre = "Arturo",
edad =18;
if( edad === "18" ){
document.write("OK");
} else {
document.write("false");
}
