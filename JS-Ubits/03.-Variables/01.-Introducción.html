Variable

La variable es un espacio de memoria donde almacenamos información.

Declarar una variable

En JavaScript una variable se declara usando:

  1. var (crea variable globales) - Evítalo
  2. let (crea variables locales) - valores que cambian
  3. const - cuando el valor no va a cambiar

Reglas para nombres de variables

  • Deben empezar con letra, _ o $
  • No pueden empezar con número
  • No pueden usar palabras reservadas (if, for, function, etc.)
  • Son sensibles a mayúsculas (edad ≠ Edad)

Ámbito de la variable

Las variables tienen un ámbito o alcance, lo correcto es que las variables existan localmente, sólo en el bloque en el que fueron definidas.
Sin embargo, las variables que se declaran con var son GLOBALES, las cuales existen en toda la ejeción del programa.

  • Var: Crea variables globales
  • Let: Crea variables locales
