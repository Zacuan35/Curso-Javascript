VAR - Genera variables globales.

En el siguiente ejemplo la variable message esta declarada dentro del bloque de contenido.
En teoria sólo debería existir dentro de ese bloque de contenido, pero si la mandamos llamar afuera de este, sigue existiendo, existe a lo largo de todo el documento javascript.
Una variable no debería existir ni ser invocada fuera del bloque donde fué definida.

Tener una variable global es como si en el ejemplo anterior tuviéramos declarada la variable fuera del bloque de contenido.

Elevación de variables (Hoisting)

Al declarar una variable con var, independientemente de donde la declaremos JS eleva al inicio del documento.
