Null-Undefined

La variable carece de valor o el valor esta ausente

Undefined

Se genera cuando a una variable no se le define o asigna un valor, o sencillamente no existe.
Aparecen como errores que generalmente no esperamos y que son despistes del programador.

En el siguiente ejercicio se declara una variable pero no se le asigna un valor Undefined indica que no se ha inicializado una variable y que el valor esta ausente

Null

Es un valor especial que indica la ausencia de un valor
Se le asigna un valor vacío con la palabra reservada Null de forma intencional a la variable.
Habrá ocasiones en las que será necesario que nuestras variables comiencen con un valor ausente o vacío en nuestro flujo de programación
