TIPOS DE OPERADORES

OPERADORES ARITMËTICOS
  • +, - : Suma y resta
  • * , / : Multiplicación y división
  • % : Módulo, devuelve el residuo de una división
OPERADORES RELACIONALES
  • >< : Mayor que, menor que
  • >=<= : Mayor o igual que, Menor o igual que
  • == : Igualdad débil (valor)
  • === : Igualdad fuerte (valor y tipo)
  • != : Desigualdad débil (valor)
  • !== : Desigualdad estricta (valor y tipo)
OPERADORES DE TIPOS DE DATOS
  • typeof : Devuelve el tipo de dato (como texto)
  • instanceof : Verifica si un objeto es instancia de una clase
OPERADORES LÓGICOS
  • && : (AND - Y) Todas las condiciones deben cumplirse
  • || : (OR - O) Al menos una de las condiciones debe de cumplirse
  • ! : (NOT) Negación
OPERADORES DE INCREMENTO O DECREMENTO
  • i++, i-- : Operador de incremento y decremento
  • i+= n : Suma n a i (suma y asigna)
  • i-= n : Resta n a i (resta y asigna)
