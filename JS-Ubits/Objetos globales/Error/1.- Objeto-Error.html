Objeto error

¿Que és?

  • Error es un objeto nativo
  • Representa fallos en el código / programa
  • Tiene propiedades útiles
  • Se maneja con try...catch
  • Se puede lanzar manualmente

Sirve para:

  • Describir qué salió mal
  • Mostrar mensajes de error
  • Manejar fallos sin que el programa se rompa
