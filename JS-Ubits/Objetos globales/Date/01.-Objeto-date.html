Objeto Date

  • El objeto Date de JavaScript sirve para trabajar con fechas y horas.
  • Obtener la fecha actual, crear fechas específicas, compararlas, formatearlas, etc.
  • Formato recomendado: YYYY-MM-DD (ISO 8601).
  • Date usa la fecha y hora (zona horaria) del sistema

- Los meses empiezan en 0
0 = enero
11 = diciembre
