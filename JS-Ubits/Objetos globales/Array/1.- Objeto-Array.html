Objeto array

¿Qué es un Array?

  • Es un objeto especial que permite almacenar varios valores dentro de una sola variable
  • Organza los valores en una lista ordenada de elementos
  • Cada elemento ocupa una posición numérica llamada índice, la cual comienza desde 0.
  • Tiene métodos propios
  • Hereda de Object
  • Técnicamente Array es un objeto y no un tipo de dato primitivo.

¿Por qué se dice que es un objeto?

Array es un objeto especializado para trabajar con listas porque:

  • Tiene propiedades (length)
  • Tiene métodos
  • Hereda de Object.prototype
