Variable

La variable es un espacio de memoria donde almacenamos información.

Declarar una variable

En JavaScript una variable se declara usando:

  1. var - Evítalo
  2. let - valores que cambian
  3. const - cuando el valor no va a cambiar

Reglas para nombres de variables

  • Deben empezar con letra, _ o $
  • No pueden empezar con número
  • No pueden usar palabras reservadas (if, for, function, etc.)
  • Son sensibles a mayúsculas (edad ≠ Edad)
