STRING - Cadenas de caractéres

los strings son un tipo de dato primitivo que se usa para representar texto.
Un string es una cadena de caracteres (letras, números, símbolos o espacios).

¿Cómo se escriben los string

En JS puedes escribirlos de tres formas:

  • Comillas dobles: "Hola mundo"
  • Comillas simples: 'Hola mundo'
  • Backticks: `Hola mundo`

Un string puede contener cualquier carácter:

  • "Texto"
  • "123"
  • "!@#$%"
  • "Hola 2026"
