FUNCIONES

¿Qué es?

Una función es un bloque de código autocontenido, este bloque tiene un conjunto de instrucciones (líneas de código) que ejecutan ciertas acciones.

¿Cómo funciona?

Una función primero se define una vez y después se ejecuta en cualquier momento.

Sintaxis

  • Function: palabra clave que indica que se va a declarar una función.
  • Nombre de la función: es el identificador que servirá para llamarla después; por ejemplo sumar.
  • Paréntesis (): Encierran los parámetros, siempre debe ir aunque no haya parámetros.
  • Parámetros (a, b) : son variables que recibe la función.
  • Cuerpo de la función {...}: Es el bloque de código que se ejecuta cuando se llama la función.
  • Return: el valor que la función devuelve (opcional). Si no hay return, devuelve undefined, ejemplo return a + b;
