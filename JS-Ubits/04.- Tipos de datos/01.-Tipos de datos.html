Tipos de datos

Existen diferentes tipos de datos que una variable puede almacenar, en JS se subclasifican en dos tipos.

Primitivos

Se accede directamente al valor de la variable

  • String - Cadenas de texto
  • Numbers - Números
  • Booleans - True o false

Complejos

Se accede a la referencia del valor, a los índices o coordenadas del valor,
corchete [] para arreglos, y notación de punto . para los arreglos.

  • Functions - Funciones
  • Arrays - Arreglos
  • Objects - Objetos

DATOS PRIMITIVOS

Se accede directamente al valor de la variable

DATOS COMPLEJOS

Se accede a la referencia del valor

Funciones

Las funciones primero se declaran y después se llaman

Arreglos - Listas de datos

  • La lista puede contener diferentes tipos de valores
  • La lista tienen posiciones llamadas índices, estas comienzan con el número 0.
  • Estas posiciones nos sirven como referencia para acceder al valor.
  • El último elemento de una lista de arreglos no lleva coma

Objeto

Es una colección de propiedades y valores
