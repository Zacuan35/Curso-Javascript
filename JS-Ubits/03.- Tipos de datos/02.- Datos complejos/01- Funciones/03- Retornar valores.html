Retornar valores

1.- Devolver o retornar un valor

Opcionalmente una función puede devolver un valor para lo cual se usa return.
Return sirve para devolver el valor del cuerpo de la función hacia la línea donde es invocada

¿Cómo funciona el return?

Cuando el código dentro de una función encuentra un return, el valor sale de la función y se va a donde fué declarado y lo demás, es ignorado.
