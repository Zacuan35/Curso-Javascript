Variables

¿Qué son las variables y los identificadores?

  • Las variables son contenedores para almacenar valores de datos.
  • Un identificador es el nombre que le das a una variable, y deben ser únicos.

Reglas para nombrar identificadores

  • Debe comenzar con una letra, _, o $
  • Puede contener dígitos después del primer carácter.
  • No puede ser una palabra clave reservada (let, const, if, etc.)
  • Son sensibles a mayúsculas y minúsculas
  • Los guiones no están permitidos en JavaScript. Están reservados para restas.

¿Cómo declarar variables?

  • Primero se define la variable (x).
  • Después, a la varible se le asigna el valor (6).
