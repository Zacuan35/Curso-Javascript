Usando window.alert()

Con la palabra clave window

Puede utilizar un cuadro de alerta para mostrar datos

Sin la palabra clave window

Puedes omitir la palabra clave window.

En JavaScript, el objeto ventana es de ámbito global.
Esto significa que las variables, propiedades y métodos pertenecen al objeto ventana por defecto, por lo cual la palabra clave window es opcional.
